# Model checkpoints

Place model checkpoint files in this folder before using the browser interface.

The interface recognises these filenames:

- `yolo8n.pt`, `yolo8s.pt`, `yolo8m.pt`, `yolo8l.pt`
- `yolo10n.pt`, `yolo10s.pt`, `yolo10m.pt`
- `yolo11n.pt`, `yolo11s.pt`, `yolo11m.pt`, `yolo11l.pt`, `yolo11x.pt`

Checkpoint files are not stored in Git because they are large. Copy trained weights into this folder and rename them to the model option they belong to.
